# Briefing do Projeto

**Nome do Projeto**: PWA de Agendamento de Consultas

**Objetivo Principal**: Desenvolver uma aplicação web progressiva (PWA) para agendamento e visualização de consultas médicas, com layout otimizado para dispositivos móveis.

**Tecnologias Principais**:
- Node.js
- Express.js
- Handlebars (templates)
- CSS (estilização)
- PWA (Service Workers, Manifest)

**Requisitos de Layout**:
- Deve seguir o design da imagem de referência
- Principalmente cores azuis, com branco como cor secundária
- Interface limpa e minimalista
- Responsiva para dispositivos móveis

**Funcionalidades Iniciais (Frontend)**:
- Exibição de consultas agendadas
- Cards de consulta com informações detalhadas
- Navegação básica (header e footer)
- Suporte offline básico via PWA
